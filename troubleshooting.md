# Troubleshooting

Common Issues & Fixes

**`Command not found: neuroncli`**

* Ensure NeuronCLI is installed correctly.
*   Try reinstalling:

    ```
    pip install --upgrade neuroncli
    ```

**`Agent execution failed`**

* Check logs at `~/.neuroncli/logs/`
* Ensure that your agent is correctly configured.
